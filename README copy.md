# Minimal Roommate App Starter

This is the simplest reasonable starter framework for the Sustainable Roommate Agreement project.

## Files
- `package.json` - project dependencies
- `server.js` - all backend code, database setup, and API routes
- `public/index.html` - all frontend HTML, CSS, and JavaScript in one file
- `README.md` - setup instructions

## How to run
1. Open terminal in this folder
2. Run `npm install`
3. Run `npm start`
4. Open `http://localhost:3000`

- Create one GitHub repo
- Upload these files
- Each person makes their own branch
- Each person edits only their section first
- Merge into main when tested
